配置

Odin 允许用户根据个人或组织的需要配置工具的运行参数和行为。

代理设置

  1. 配置代理

    • 如果用例执行过程中需要通过代理服务器发包，在此进行配置。

  2. 输入代理信息

    • 提供代理服务器的地址、端口以及可能需要的认证信息。

  3. 保存设置

    • 保存代理设置。

系统设置

  1. 自动升级设置

    • 选择是否希望Odin工具联网自动检查并安装新版本。

  2. 超时时间设置

    • 设置用例执行的超时时间，以避免无限期的等待。

  3. 并发数设置

    • 根据系统资源和网络条件，设置用例执行的并发数。

  4. 保存系统设置

    • 调整完毕后，保存系统设置以立即生效。

WAF设置

  1. WAF验证规则配置

    • 根据所使用的Web应用防火墙（WAF）产品，配置相应的验证规则。

  2. 规则编辑

    • 如果需要，可以手动添加或修改WAF的验证规则。

  3. 保存WAF设置

    • 测试成功后，保存WAF设置。

注意事项

  • 在修改任何设置后，确保保存以使更改生效。
  • 错误的代理设置可能导致用例无法正常执行，所以请仔细检查代理信息。
  • 系统设置中的超时时间和并发数应根据实际的系统性能进行调整，以避免资源耗尽。
  • WAF设置直接影响到安全防护的效果，请在专业人员的指导下进行配置。

通过以上步骤，用户可以根据自己的需求对Odin工具进行个性化配置，确保工具能够以最适合组织的方式来运行。
