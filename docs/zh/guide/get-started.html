快速上手

WARNING

Odin 目前仍处于内测阶段。你可以使用它来测试 WAF 防护的有效性，但是部分功能还在持续开发中。因此，在每次更新版本之后，请一定要仔细阅读 更新日志open in new window。

安装

依赖环境

  • Windows AMD64/ARM64 10 及以上版本
  • Linux AMD64/ARM64 Glibc >= 2.35
  • MacOS AMD64/ARM64 12.5.1 及以上版本

TIP

  • Odin 提供编译后的二进制文件，可直接双击启动。
  • MacOS 需要将 Contents/MacOS/Odin 添加可执行权限：chmod +x Contents/MacOS/Odin

功能结构

启动项目后，侧边栏的功能如下：

  1. 统计

    • 数据汇总：提供项目关键数据的汇总视图。
    • 图表展示：通过图表形式直观展示统计数据。

  2. 用例库

    • 用例管理：允许用户创建、编辑和删除用例。

  3. 场景库

    • 场景创建：用户可以定义不同的测试场景。

  4. 检测任务

    • 任务配置：设置检测任务的参数和选项。
    • 任务启动：启动配置好的检测任务。

  5. 验证结果

    • 结果查看：展示检测任务的输出结果。
    • 结果分析：对检测结果进行深入分析。

  6. 历史结果

    • 历史数据查询：允许用户回溯和查询历史测试结果。
    • 数据对比：比较不同时间点的测试结果。

  7. 设置

    • 代理设置：WAF 发包的代理配置。
    • 系统设置：用例超时时间以及并发数配置。
    • WAF 设置：允许用户创建、编辑和删除 WAF 规则。

  8. 关于我们

    • 公司信息：提供公司的基本信息和联系方式。

开始使用

创建检测任务

点击用例或者场景，然后点击添加按钮选择要测试的用例或场景。

请为您的任务指定一个名称，并提供需要进行安全测试的目标URL。建议目标地址应位于Web应用防火墙（WAF）的保护范围内，这样Odin所模拟的攻击流量就能够通过WAF设备，从而更准确地评估WAF的安全防护效果。
